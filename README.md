# 🌬️ Terra Ventos

Bem-vindo ao repositório oficial da **Terra Ventos**! Este projeto é uma landing page moderna e interativa desenvolvida com React e Vite, focada em apresentar projetos imobiliários, estudos de ventos e oportunidades de investimento na região.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores e mais modernas tecnologias do ecossistema front-end:

*   **[React](https://reactjs.org/)**: Biblioteca JavaScript para criação de interfaces de usuário.
*   **[Vite](https://vitejs.dev/)**: Ferramenta de build super rápida que otimiza o desenvolvimento.
*   **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para estilização ágil e responsiva.
*   **Lucide React**: Conjunto de ícones limpos e consistentes.

## ✨ Funcionalidades Principais

*   **Design Responsivo**: Adaptado para telas de celulares, tablets e desktops de forma fluida.
*   **Apresentação de Empreendimentos**: Catálogos dinâmicos de imóveis, terrenos e projetos ("Bitupitá", entre outros).
*   **Estudos dos Ventos**: Seção dedicada à análise da força dos ventos na região.
*   **Busca de Imóveis**: Widget interativo para facilitar a navegação do usuário.
*   **Seções de Contato e Consultoria**: Formulários e chamadas para ação focados em conversão.

## 📦 Como executar o projeto localmente

Siga o passo a passo abaixo para rodar o projeto na sua máquina:

### Pré-requisitos
Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.

### Passos

1. Clone este repositório:
```bash
git clone https://github.com/leonardorimes/siteterraventos.git
```

2. Acesse a pasta do projeto:
```bash
cd siteterraventos
```

3. Instale as dependências:
```bash
npm install
```

4. Execute o servidor de desenvolvimento:
```bash
npm run dev
```

5. O projeto estará rodando, geralmente em `http://localhost:5173`.
